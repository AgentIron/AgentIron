# AgentIron Development Notes

## Related Repositories

`iron-core` and `iron-providers` are maintained by the same author as this project.
Bugs or improvements needed in those crates should be filed as GitHub issues:

- **iron-core**: https://github.com/AgentIron/iron-core/issues
- **iron-providers**: https://github.com/AgentIron/iron-providers/issues

Do not work around upstream bugs silently — file an issue so it gets fixed at the source.
